Next.js + Supabase + Tailwind starter created for Lizzy.

Features:
- Next.js App Router
- TailwindCSS styling
- Supabase Auth (email + password)
- Protected dashboard route
- Placeholder AI API route for future integration

Setup:
1. Run: npm install
2. Copy .env.example to .env.local and fill SUPABASE URL + ANON KEY
3. Run: npm run dev
